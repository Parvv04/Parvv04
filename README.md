## 🚀 Featured Projects

### MailMinder

* Gmail meeting reminder system
* Built using Python, FastAPI, Electron, and Gmail API
* Uses NLP and regex to extract meeting details and send desktop notifications

### Smart Health Monitoring System

* Real-time health monitoring using computer vision
* Built with Python, OpenCV, MediaPipe
* Detects drowsiness, poor posture, and unhealthy screen-time habits

### Gesture Controlled Assistive Mobility & Communication System

* Sensor-based wearable glove for gesture recognition
* Uses ESP32, Flex Sensors, MPU6050, and Machine Learning
